# Adir-Property-Management-System
Adir is a comprehensive Building Management System designed to streamline property operations, apartment tracking, and project lifecycle management. This project demonstrates the power of Salesforce Declarative Tools combined with Apex programming to solve real-world business challenges.

🚀 Key Features & Implementation 

Data Modeling:
Architected a relational database schema using Custom Objects (Buildings, Apartments, Projects) with Master-Detail and Lookup relationships.

Automation Excellence: Leveraged Record-Triggered Flows to automate status updates and task assignments.

Advanced Logic: Implemented Validation Rules and Roll-up Summaries to ensure data integrity and real-time financial reporting.

Security & UX: Designed custom Lightning Record Pages and implemented Permission Sets to maintain a "Security-First" approach.

🛠️ Tech Stack
Platform: 
Salesforce Service Cloud

Logic: Apex (Triggers & Classes), Flow Builder

UI/UX: Lightning Web Components (LWC), App Builder

Testing: Manual UAT & Unit Testing (Apex Test Classes)
